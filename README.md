# Portafolio

Portafolio bilingüe construido con Astro y Preact.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- `src/pages/references.astro` muestra los frames del ZIP para trazabilidad visual.
- El menú responsive vive en Preact y solo se hidrata donde hace falta.
